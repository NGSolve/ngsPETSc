ngsPETSc
------------------

ngsPETSc is a PETSc interface for NGSolve.

Installation
-----------------
To install ngsPETSc you need to clone the GitHub repository, and then you can install it using pip.
::
    git clone https://github.com/UZerbinati/ngsPETSc.git
    cd ngsPETSc
    pip install .
